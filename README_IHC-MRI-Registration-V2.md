## 🚀 Release v2.0.0

We are excited to announce the release of **version 2.0.0** of the *IHC-MRI-Registration* software!

### ✨ Highlights

Version 2.0.0 introduces significant improvements, including:

- ⚡ **Enhanced performance** through a complete migration from **Tkinter** to **PyQt5**
- 🧠 **Various new functionalities** for a more precise and streamlined co-registration workflow
- 🖥️ **New intuitive graphical user interface (GUI)**
- 📘 **Built-in documentation**:
  - Available directly on the GUI main page
  - Additional detailed guidance within the *Non-Linear Registration* step
